# Cardio Lit Monitor 🫀

Automated weekly literature monitor for cardiovascular genomics and NDD research.

**Live site:** https://jlmthompson.github.io/cardio-lit-monitor/

## What it monitors
- 🔴 **Gene Watch** — new papers on CHD-NDD overlap genes and personal research genes
- 💙 **PRS Watch** — polygenic risk score papers (CHD, DCM, cardiovascular, NDD, cardiometabolic)
- 🟢 **Topic Watch** — episignatures, cardiac methylation, epigenetic age, multi-omics
- 🆕 **PGS Catalog** — new polygenic scores added for relevant traits

## Schedule
Runs every Monday at 7 AM Sydney time. Email digest sent to work address.

## Config
Edit `data/watch_config.json` to add/remove genes, queries, or topics.
